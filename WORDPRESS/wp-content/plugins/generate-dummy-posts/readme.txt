== Description ==
With Generate Dummy Posts plugin, You can create dummy posts in just one click. Developers and designers often used dummy text to create or develop blogging wordpress website and it takes time to create several posts one by one. This plugins will save your time.

[youtube https://www.youtube.com/watch?v=XAOwKVNa3iU]

* Simple setup.
* Generate posts.
